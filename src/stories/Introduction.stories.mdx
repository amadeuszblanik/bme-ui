# Welcome to BME-UI

BME-UI is a React component library for building user interfaces.

## Installation

Install the package using npm:
```
npm install bme-ui
```

## Usage

Import the component you want to use:
```
import { Button } from '@bme-ui';
```

## Documentation

You can find the documentation [here](https://github.com/amadeuszblanik/bme-ui).
